Fuel-php-twitter-bootstrap

Simple boilerplate to help you quickly create Fuel PHP projects using Jquery Mobile, Twitter Bootstrap, FuelPHP + more created by Eric The Robot

Download this project as a .zip file Download this project as a tar.gz file

Welcome to a Simple Boilerplate using FuelPHP / Twitter Bootstrap / jQuery Mobile / HTML5

The purpose of this open-source project is to help developers quickly create powerful web and mobile applications using tools that are proven to work in high performance environments. To get started, simply clone the repository below:

$ git clone https://github.com/StopBrain/fuel-php-twitter-bootstrap.git
$ cd ~/Sites/fuel-php-twitter-bootstrap

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Versions & Resources

  • Fuel PHP v1.2 - fuelphp.com
  • Twitter Bootstrap 2.0.4 - twitter.github.com/bootstrap
  • jQuery v1.7.1 - jquery.com
  • jQuery Mobile v1.1.0 - jquerymobile.com
